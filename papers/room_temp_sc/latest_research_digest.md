## 1. LK-99 后续验证与室温常压超导声称的独立复现

### 1.1 LK-99（Pb₁₀₋ₓCuₓ(PO₄)₆O）声称的终结

| 字段 | 内容 |
|------|------|
| **标题** | 室温常压超导材料 LK-99 的声称与系统性证伪 |
| **原始声称** | Lee et al. (2023), arXiv:2307.12008 / 2307.12037 |
| **关键验证论文** | 多篇独立团队发表于 Nature Materials, arXiv 等 |
| **arXiv号** | 2307.12008（原始），后续大量反驳论文 |
| **关键结论** | LK-99 的"悬浮"现象源于铁磁性而非迈斯纳效应；零电阻未被复现；其半导体行为可归因于 Cu₂S 杂质。全球数十个独立实验室未能复现超导特征。 |

**详细总结：**

LK-99 在 2023 年 7 月引发全球轰动，但随后数月内迅速被证伪：

- **Nature Materials (2024)**：多个团队确认 LK-99 不存在体超导性。其磁性响应与 Cu₂S 的杂质铁磁性一致，而非超导体的抗磁性。
- **电阻测量**：无独立团队观察到零电阻。所有样品均表现为半导体或绝缘体行为。
- **结构分析**：X射线/中子衍射显示其为普通绝缘体结构，无任何超导能隙特征。
- **理论解释**：DFT 计算表明 LK-99 的电子结构不支持强电子-声子耦合，也没有平带或范霍夫奇点（VHS）来驱动非常规超导。

> **结论**：LK-99 已被科学界广泛认定为**非超导材料**。其原始声称是实验误读（杂质效应）与过早宣传的结果。

### 1.2 Ranga Dias 室温超导声称的彻底崩塌

| 字段 | 内容 |
|------|------|
| **标题** | Evidence of near-ambient superconductivity in a N-doped lutetium hydride |
| **作者** | Dasenbrock-Gammon et al. (Ranga Dias 团队) |
| **期刊** | Nature (2023, 已撤稿) |
| **arXiv号** | 10.1038/s41586-023-05742-0（DOI） |
| **关键结论** | 声称在 1 GPa、294 K（21°C）下实现超导。后因数据伪造和学术不端被 Nature 撤稿。 |

**后续发展：**

- **2022 年 9 月**：Dias 团队首篇室温超导论文（CSH 材料，15°C @ 267 GPa）被 Nature 撤稿，理由是数据处理不当（背景减除方法"非标准"）。
- **2023 年 3 月**：Dias 再次声称在氮掺杂氢化镥（Lu-N-H）中实现近室温超导（1 GPa 下 20°C），发表于 Nature。随后被同行质疑数据篡改。
- **2024 年 3-4 月**：Nature 发表独家调查，披露罗切斯特大学调查委员会认定 Dias 存在**系统性学术造假**。证据包括：原始数据被伪造、剽窃博士论文、误导合作者。罗切斯特大学计划解雇 Dias。
- **2024 年**：Dias 的博士论文和 PRL 论文也被揭露存在数据操纵。Physical Review 启动调查。

> **结论**：Ranga Dias 的两篇 Nature 室温超导论文均已被撤稿，其学术声誉彻底破产。**这是近年来凝聚态物理领域最严重的学术造假事件之一**。

## 2. 铜氧化物（Cuprate）超导新进展

### 2.1 条纹相（Stripe Phase）与 d-波超导的竞争：DMRG 计算新突破

| 字段 | 内容 |
|------|------|
| **标题** | Phase Diagram, d-Wave Superconductivity, and Pseudogap of the t-t'-J Model at Finite Temperature |
| **作者** | 多个团队（DMRG 计算） |
| **期刊/来源** | arXiv:2211.06322v4 (2024 更新); Phys. Rev. Lett. 132, 066002 (2024) |
| **arXiv号** | arXiv:2211.06322; arXiv:2402.17305; arXiv:2311.15092 |
| **关键结论** | 在 t-t'-J 模型中，当考虑次近邻跃迁 t' 时，DMRG 计算发现条纹序被抑制，出现稳定的 d-波超导相。有限温度下存在赝能隙。 |

**详细总结：**

铜氧化物高温超导的微观机理（是否可用 Hubbard/t-J 模型描述）是凝聚态物理的核心难题之一。2024-2025 年，基于大规模 DMRG（密度矩阵重整化群）和 DQMC（行列式量子蒙特卡洛）的研究取得重要进展：

- **Phys. Rev. Lett. 132, 066002 (2024)** — Lu, Chen, Zhu, Sheng, Gong：
  - 在空穴掺杂的方格子 t-J 模型中，发现了**涌现超导性与竞争电荷序**的共存。
  - 使用 DMRG 在宽圆柱几何中计算，发现超导关联与 CDW 关联呈竞争关系。

- **arXiv:2402.17305 (2024)** — Sun, Ying, Scalettar, Mondaini：
  - 研究了各向同性配对系统中条纹与电荷 π-相移的涌现。
  - 发现条纹相可以与超导基态共存。

- **arXiv:2311.15092 (2023)** — Chen, Haldane, Sheng：
  - 在方格子 t-J 模型中发现了 **d-波和配对密度波（PDW）超导**。
  - 表明 Hubbard 模型可能确实包含高温超导的要素。

- **arXiv:2501.15709 (2025)** — Devereaux 等综述：
  - 总结了 DMRG 在 t-J 和 Hubbard 模型中的最新结果。
  - 关键发现：对于 t' ≤ 0（模拟铜能带结构），CDW/条纹序倾向于战胜超导；对于 t' > 0，超导更稳定。
  - 挑战：Friedel 振荡、边界效应、有限尺寸效应仍影响外推至二维极限。

> **意义**：这些数值结果首次在微观模型中**自洽地**重现了 d-波超导、条纹序和赝能隙，为铜氧化物超导的"标准模型"（Hubbard/t-J 模型）提供了强有力的支持。但二维极限下的长程序仍待澄清。

### 2.3 铜氧化物超导的"缺失环节"：什么决定了序的竞争？

当前核心问题（2024-2025 综述）：

1. **条纹 vs. 超导**：在 t' < 0 的模型中，为什么实验上铜氧化物表现出稳健超导，而数值上条纹似乎更强？可能的答案：
   - 二维极限下长程关联的微妙竞争
   - 层间耦合、无序、声子等被忽略的因素
   - 当前 DMRG 的圆柱宽度和长度仍有限

2. **赝能隙的起源**：DQMC 计算显示在 T ~ J/k_B 时存在波动条纹和 d-波配对 susceptibility，但有限温度相变难以确定。

3. **最佳掺杂的奥秘**：DMRG 在 t'/t ≈ 0.17、δ ≈ 0.12 附近发现最大的超导关联，与实验最佳掺杂区域一致。

## 3. 氢化物高压超导新数据

### 3.1 LaH₁₀：接近室温的纪录保持者（2019-2025）

| 字段 | 内容 |
|------|------|
| **标题** | Superconductivity at 250 K in lanthanum hydride under high pressures |
| **作者** | Drozdov, Kong, Minkov, Besedin, Kuzovnikov, Mozaffari, Balicas, Balakirev, Graf, Prakapenka, Greenberg, Knyazev, Tkacz, Eremets |
| **期刊** | Nature 569, 528-531 (2019) |
| **arXiv号** | arXiv:1812.01561 |
| **关键结论** | 在约 170 GPa 下，LaH₁₀（Fm-3m 结构）出现 Tc ≈ 250-260 K 的超导。通过零电阻、同位素效应和磁场抑制确认。这是目前实验验证的最高 Tc 记录。 |

**后续验证：**

- 至少三个独立团队（包括美国、中国、欧洲）复现了 LaH₁₀ 的超导特征。
- 压力-温度相图：Tc 在 150-200 GPa 范围内达到峰值 250-260 K，在 136 GPa 处仍保持约 246 K。
- **挑战**：电阻下降幅度通常不完全到零，归因于样品不均匀性和接触问题；完全的抗磁性测量因样品尺寸小而困难。

### 3.3 光增强超导：Floquet 工程的新方向（2024-2025）

| 字段 | 内容 |
|------|------|
| **标题** | Floquet-engineered superconductivity in hydrides ( Floquet-DFPT 研究 ) |
| **作者** | Xie 等 |
| **期刊** | npj Computational Materials (2025) |
| **关键结论** | 使用 Floquet-DFPT（密度泛函微扰理论）模拟，发现激光泵浦可在 LaH₁₀ 中增强电子-声子耦合。预测在 150 GPa 下，强泵浦可将 Tc 从 249 K 提升至 285 K；在 200 GPa 下，从 240 K 提升至 301 K——接近室温。 |

**机制：**

- 激光场周期性地驱动电子系统，改变费米面附近的态密度和电子-声子矩阵元。

- 挑战：低于 210 GPa 时，强晶格非谐性使模拟不可靠；实验验证尚未实现。

> **意义**：如果 Floquet 增强超导在实验上可行，可能为"动态室温超导"开辟新路径——无需新材料，而是利用光场调控现有材料。

## 4. 超导理论最新发展

### 4.1 BCS 理论的扩展与非常规配对机制

#### 4.1.1 d-波配对与自旋涨落驱动

| 字段 | 内容 |
|------|------|
| **标题** | 铜氧化物中的 d-波超导：自旋涨落与条纹相 |
| **作者** | 多个理论组（Scalapino, Kivelson, Devereaux, Sheng, Jiang 等） |
| **关键结论** | 在 t-t'-J/Hubbard 模型中，自旋涨落通过交换反铁磁相互作用诱导 d-波配对。条纹相（一维电荷密度波）与超导竞争，但可能共存。 |

**核心进展（2024-2025）：**

1. **DMRG 在宽圆柱上的突破**：
   - 宽度为 8 的圆柱上，DMRG 首次在 t'/t > 0 时观察到**长程 d-波超导关联**。
   - 在 t'/t < 0（更接近铜氧化物）时，条纹序更强，但超导关联仍存在于局部尺度。

2. **配对密度波（PDW）**：
   - Chen, Haldane, Sheng (arXiv:2311.15092, 2023) 发现 PDW 与 d-波超导在 t-J 模型中可能共存。
   - PDW 的特征是配对振幅的空间调制，周期与条纹序一致。

3. **自旋-电荷分离与相位弦效应**：
   - Weng, Sheng, Chen, Ting 提出的相位弦理论（Phase String Theory）继续发展。
   - 在掺杂反铁磁体中，空穴运动伴随不可压缩的相位弦，影响配对和输运。

### 4.2 阿布里科索夫-陈（Abrikosov-Chern）超导态（2024）

| 字段 | 内容 |
|------|------|
| **标题** | Making s-wave superconductors topological with magnetic field |
| **作者** | Daniil Antonenko 等 |
| **来源** | arXiv:2409.15266 (2024; v2 Nov 2025) |
| **关键结论** | 2D s-波超导体在磁场下形成阿布里科索夫涡旋格子时，可变为拓扑非平庸态。拓扑数为偶数（不支持 Majorana），但边缘态数目变化可通过隧穿谱学探测。 |

**意义：**

- 揭示了从整数量子霍尔态（上临界场以上）到拓扑平庸 s-波超导（低场）的演化。
- 拓扑转变表现为边缘模数目的变化，可通过热输运或自旋输运测量。
- 为理解传统 s-波超导体的拓扑相变提供了新框架。

## 5. 与 Sylva 超导形式化证明相关的数学工具

### 5.1 拓扑超导与陈数的形式化验证

#### 5.1.1 陈数的物理可观测性：Winding Number 分解

| 字段 | 内容 |
|------|------|
| **标题** | Chern number as an observable in topological superconductors |
| **作者** | 未明确列出 |
| **来源** | arXiv:1209.5115 |
| **关键结论** | 陈数可计算为物理可观测的 winding number。在多分量系统（如自旋、轨道、子晶格）中，陈数可分解为各伪自旋分量 winding number 之和。其奇偶性可通过密度测量区分是否支持 Majorana 模。 |

**对 Sylva 项目的意义：**

- 陈数的分解性质为**形式化证明**提供了模块化策略：分别证明各分量的 winding number，再求和得到总陈数。
- 奇偶性判据（parity check）提供了一种"弱形式化"路径：无需完整陈数计算，仅需证明其奇偶性即可判断 Majorana 支持性。

| **作者** | S. Lahiri 等 |
| **期刊** | Nature / Scientific Reports (2024), 被引 12 次 |
| **关键结论** | 在能带设计的 Haldane 模型中，通过调控最近邻跃迁振幅比 t1/t，使系统从 Chern 绝缘体（C=1）转变为二阶拓扑绝缘体（SOTI，C=0）。SOTI 相具有零能角态，体极化（bulk polarization）为拓扑不变量。 |

**拓扑不变量演化：**

- 当 ξ = t1/t < 2 时，Chern 数 C = 1（一阶拓扑相）。
- 当 ξ > 2 时，Chern 数 C = 0，但体极化 p_y = a/2（二阶拓扑相）。
- 这种"陈数归零但拓扑非平庸"的现象对理解拓扑超导的**高阶拓扑**至关重要。

### 5.3 拓扑绝缘体与超导体的分类：Kitaev 周期表

| 字段 | 内容 |
|------|------|
| **标题** | Periodic table for topological insulators and superconductors |
| **作者** | Alexei Kitaev |
| **来源** | AIP Conference Proceedings 1134, 22 (2009) |
| **关键结论** | 基于对称性（时间反演、粒子-空穴、手性）和维度，对拓扑绝缘体和超导体进行十重分类。不同维度和对称性下，拓扑不变量可为 Z, Z₂, 或 0。 |

**对 Sylva 形式化的映射：**

| 维度 | 对称类 | 拓扑不变量 | 物理实例 | 形式化难度 |
|------|--------|------------|----------|------------|
| 2D | A (无对称) | Z (Chern) | 整数量子霍尔 | ★★☆ |
| 2D | AIII (手性) | Z | 拓扑超导体 | ★★★ |
| 1D | AIII | Z | 拓扑超导线 | ★★☆ |
| 1D | D (粒子-空穴) | Z₂ | Majorana 链 | ★★☆ |
| 3D | AII (TR²=-1) | Z₂ | 拓扑绝缘体 | ★★★ |

> **建议**：Sylva 的形式化证明可优先从 **1D Class D（Z₂）** 或 **2D Class A（Z）** 入手，因为这些情形拓扑不变量的计算最为成熟（winding number / Chern number 的积分公式）。

### 5.4 拓扑不变量计算的形式化策略

基于上述文献，为 Sylva 项目建议以下形式化路线：

**Phase 1: 基础框架**
1. 在 Lean 中定义 Berry 联络和 Berry 曲率（纤维丛上的联络形式）。
2. 证明 Berry 相位的规范不变性和几何性（非绝热演化中的相位）。
3. 形式化 Brillouin 区上的周期边界条件。

**Phase 2: 陈数计算**
1. 将陈数定义为 Berry 曲率在 Brillouin 区上的积分。
2. 证明其整数性（通过离散化格点和组合抵消）。
3. 形式化陈-韦伊同态：从曲率形式到示性类。

**Phase 3: 超导应用**
1. 将 Bogoliubov-de Gennes (BdG) 哈密顿量编码为对称类 D 或 AIII。
2. 证明 BdG 能带的拓扑不变量（陈数 / winding number）与准粒子激发的边界态数目对应。
3. 形式化 Majorana 零模的存在性判据（奇偶性定理）。

## 6. 结论与展望

### 6.1 实验现状总结（2024-2026）

| 目标 | 最接近的实现 | 距离室温常压 |
|------|-------------|-------------|
| 室温超导 | LaH₁₀ @ 250-260 K (需 150-200 GPa) | ~50 K 温度差，但压力鸿沟巨大 |
| 室温 + 较低压力 | LaSc₂H₂₄ 预测 316 K @ 167 GPa | 理论预测，未实验验证 |
| 室温常压 | **无可靠验证** | Dias 声称被证伪，LK-99 被证伪 |
| 光增强动态超导 | LaH₁₀ 预测 285-301 K @ 150-200 GPa | 理论预测，实验挑战巨大 |

### 6.2 理论前沿

1. **铜氧化物微观模型**：DMRG 在 Hubbard/t-J 模型中首次自洽重现 d-波超导，但二维极限下的长程序仍待解决。
2. **拓扑超导**：多分量序参量（E' 通道）和 Floquet 工程为设计新拓扑相提供了新思路。
3. **高阶拓扑**：二阶拓扑绝缘体/超导体扩展了拓扑不变量的概念，从体-边对应到体-角对应。

### 6.3 形式化数学工具就绪度

| 工具 | 成熟度 | 对 Sylva 的适用性 |

### 6.4 关键开放问题

1. **实验**：是否存在尚未发现的材料体系，可在常压下实现室温超导？
2. **理论**：Hubbard 模型是否能完全解释铜氧化物超导？条纹序与超导的精确关系是什么？
3. **计算**：如何实现更大尺度的 DMRG/DQMC 来消除有限尺寸效应？
4. **形式化**：如何将拓扑场论（如陈-西蒙斯理论）完整编码到 Lean 中，以支持更复杂的拓扑超导证明？
