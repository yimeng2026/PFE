方程 `ln(y)/y = C`（其中y = α⁻¹）的解可以用Lambert W函数表示：
```

因此，如果令：

```

## 7. 附录：Lambert W函数简介

Lambert W函数（也称Omega函数或乘积对数）定义为函数f(W) = W·e^W的反函数。

对于方程：`z = W·e^W`，其解为`W = W(z)`。

在本推导中，我们使用变换：
```
ln(y) = K·y
=> -K·y·e^(-K·y) = -K
=> -K·y = W(-K)
=> y = -W(-K)/K
```

这是求解此类对数-线性混合方程的标准技巧。
