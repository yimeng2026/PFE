# SYLVA 美观 Markdown 排版规范 v1.0

## 核心原则

- **不用表格** — 使用列表、引用、代码块替代
- **LaTeX 公式** — 所有数学用 $$ 块
- **精美符号** — 使用 Unicode 符号和 emoji
- **层次清晰** — 标题层级明确，视觉分隔

## 排版示例

### 论文条目格式

## 符号规范

### 数学符号
| 概念 | LaTeX | 示例 |
|------|-------|------|
| 描述复杂度 | `K(L)` | $K(L)$ |
| 熵间隙 | `\\Delta H` | $\\Delta H$ |
| 谱间隙 | `\\lambda_1` | $\\lambda_1$ |
| 算子 | `\\hat{H}` | $\\hat{H}$ |
| 复杂度类 | `\\mathbf{P}` | $\\mathbf{P}$ |
| 期望 | `\\mathbb{E}` | $\\mathbb{E}$ |

### 状态符号
- 🟢 完成 / 已证明
- 🟡 进行中 / 部分完成
- 🔴 未开始 / 开放问题
- ⚪ 待定 / 需评估

### 优先级符号
- 🔥 P0 — 立即执行
- ⚡ P1 — 本周执行
- 📌 P2 — 本月执行
- 📎 P3 — 长期规划
- 💤 P4 — 低优先级
