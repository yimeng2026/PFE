## 二、正反粒子引力场相同的数学证明

### 2.1 核心定理

**定理**: 正反粒子具有相同的引力场。

**证明结构**:

```
设 M 为光滑流形（粒子空间）
设 g 为度规张量（引力场的数学描述）
设 o 为定向（orientation）

正粒子: (M, g, o_left)    → 左手系定向 (+)
反粒子: (M, g, o_right)   → 右手系定向 (-)

关键性质: 
⟨X,Y⟩_g = gᵢⱼXⁱYʲ 与定向 o 无关

∵ 度规内积 ⟨·,·⟩_g 与定向选择无关
∴ 正反粒子具有相同的度规张量
∴ 正反粒子具有相同的引力场
```

### 2.2 物理诠释

| 属性 | 正粒子 | 反粒子 |
|------|--------|--------|
| **数学对象** | 左手定向流形 | 右手定向流形 |
| **电荷** | 由定向决定（见第18章） | 相反电荷 |
| **引力场** | 度规张量 g | **相同** 度规张量 g |
| **质量** | m | m（第20.8.4节证明相等） |

**重要推论**: 引力相互作用不区分物质与反物质，这与电磁相互作用形成根本对比。

## 四、中子内部结构：SU(2)⊗SU(2)⊗SU(2) 模型

### 4.1 组成成分

根据假设一，中子衰变产物决定其组成:

```
n → p + e⁻ + ν̄e

∴ 中子 = 质子 ⊗ 电子 ⊗ 反中微子
     = SU(2) ⊗ SU(2)' ⊗ SU(2)''
```

### 4.2 拓扑结构分析

**关键几何洞见** (第20.6节):

1. **质子与反中微子**具有相反定向
   - 质子: 左手系 SU(2)
   - 反中微子: 右手系 SU(2)'
   
2. **直接内积的拓扑效应**:
   ```
   SU(2) ⊗ SU(2)' ≅ S³⊗S³ / antipodal identification ≅ RP³ ≅ SO(3)
   ```

3. **磁矩减半效应**:
   - 在RP³空间中，点状质子只需旋转π/2完成一周
   - 相比自由状态的2π旋转，周期减半
   - ∴ 中子内质子磁矩 = ½ × 自由质子磁矩

### 4.3 自旋耦合几何

**120°耦合规则**:

中子自旋 = ħ/2，由三个ħ/2自旋组分合成:

```
Sₙ = Sₚ + Sₑ + Sν̄

为满足 |Sₙ| = ħ/2，三个自旋矢量在z轴投影夹角必须为120°

几何约束: Sₚ·Sₑ = Sₚ·Sν̄ = Sₑ·Sν̄ = |S|²cos(120°) = -ħ²/8
```

**图示** (第20.6.2节):
```

        ↑
        |    Sₚ(质子)
        |   /
        |  / 120°
        | /
        +-----→ Sₑ(电子)

Sν̄(反中微子)
```

## 五、宇宙膨胀的几何模型

### 5.1 物质膨胀 vs 空间膨胀

**标准宇宙学**:
```
空间本身膨胀 → 带动物质分离
问题: 需要暗能量驱动，起源不明
```

**本理论模型**:
```
粒子作为SU(2)/SO(3)群，其固有结构决定时空属性

时空间隔: ds² = -c²dt² + dx² + dy² + dz²

粒子半径演化: r(t) = c√(1 + t²)

膨胀速度: dr/dt = ct/√(1+t²) → c (当t→∞)
```

### 5.2 膨胀速度的三阶段

| 阶段 | 时间范围 | 速度特征 |
|------|---------|---------|
| **慢速期** | 0 → 1.6×10⁻⁸ s | 0 → 1475 km/s |
| **加速期** | 1.6×10⁻⁸ → 1.7×10⁻⁵ s | 1475 → 294,249 km/s |
| **光速期** | > 1.7×10⁻⁵ s | ≈ 300,000 km/s (趋于c) |

**关键数据点** (表15.5.1):
- t = 2.68×10⁻⁷ s, v = 299,977 km/s (99.99%c)
- t = 5.37×10⁻⁷ s, v = 299,994 km/s (99.998%c)

### 5.3 与观测的一致性

**无需暗能量的解释**:
- 膨胀是粒子的**内在几何属性**
- 不是时空的动力学演化
- 无需求助于负压强物质(暗能量)

## 六、质量起源与质量比计算

### 6.1 静止质量的新定义

**定义** (第20.8.1节):
```
静止质量不是基本属性，而是自旋-磁矩耦合的几何结果

m = f(自旋结构, 磁矩轨迹, 群表示)
```

### 6.2 质子-电子质量比

**理论计算** (第20.8.3节):
```
mₚ/mₑ ≈ 1836

与实验值: mₚ/mₑ = 1836.15267343(11)
```

**计算逻辑**:
1. 电子自旋磁矩 → 确定SU(2)表示
2. 质子自旋磁矩 → 确定SU(2)表示
3. 两个表示的维度差异 → 质量比

### 6.3 正反粒子质量相等

**定理** (第20.8.4节): 粒子与其反粒子具有相等的静止质量。

**证明概要**:
```
质量 m 取决于度规张量 g
正反粒子共享相同的 g
∴ m(粒子) = m(反粒子)
```
