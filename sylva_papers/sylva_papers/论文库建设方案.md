## 二、论文格式规范

### 2.1 标准结构

```
1. 标题（中英文）
2. 作者信息
3. 摘要（中英文，300字以内）
4. 关键词（5-8个）
5. 引言（研究背景、问题陈述、主要贡献）
6. 理论框架（定义、定理、证明）
7. 数值验证/实验设计
8. 结果与讨论
9. 结论与展望
10. 参考文献（标准格式）
11. 附录（可选）
```

### 2.2 格式要求

- **字体**: 正文11pt，标题18pt粗体
- **公式**: LaTeX displaymath，编号右对齐
- **图表**: 居中，标题在下方，编号连续
- **引用**: APA格式，文中[作者, 年份]，文末完整条目
- **页边距**: 上下2.5cm，左右3cm
- **行距**: 1.5倍

## 四、实时备份机制

### 4.1 备份触发条件

| 触发条件 | 备份行为 |
|---------|---------|
| 工作文件修改 | 自动复制到 backup/latest/ |
| 论文完成 | 归档到 backup/YYYY-MM-DD/ |
| 每日00:00 | 全量备份 |
| 用户指令"备份" | 立即执行全量备份 |
